# Features
- All basic features
- Bonus 1 and 2 
- semicolon and & separated commands accepted
- One header file
- Modularization: each function is defined in different file
- exit and quit commands for quitting the shell

# Commands
- cd - [. , .. , / , ~ , cwd ]
- pwd -
- echo -
- ls - [-la]
- pinfo
- remindme
- clock [-t interval -n stop ]

# Building and Running
make -j 5
./shell
